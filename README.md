\# MiniSearch Engine



A miniature search engine built using Java and Spring Boot.



This project is currently under active development.



\## Current Features



\- Web crawler

\- Inverted index

\- TF-IDF ranking

\- Spell correction

\- Snippet generation

\- Spring Boot REST API



\## Tech Stack



\- Java

\- Spring Boot

\- Maven

\- JSoup



\## Status



🚧 Work in Progress



The next milestone is building a React frontend and improving the ranking algorithm.
